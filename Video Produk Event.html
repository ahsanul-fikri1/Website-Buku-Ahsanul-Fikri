Bug Bounty Bootcamp

Rp 500.000

Video ini dari channel NahamSec membahas rekomendasi buku-buku terbaik untuk ethical hacking dan keamanan siber. Mulai dari dasar web hacking seperti "The Web Application Hacker's Handbook" yang menjelaskan serangan klien-side dan evolusi teknik hacking, hingga buku modern seperti "Bug Bounty Bootcamp" untuk mencari kerentanan web (misalnya SQL injection dan IDOR). Ada juga buku tentang API, GraphQL, dan Python untuk hacker, dengan contoh praktis dari laporan bug nyata. Cocok untuk pemula yang ingin transisi karir ke cybersecurity, menekankan pentingnya membaca untuk memperluas perspektif.

The 48 Laws of Power

Rp 200.000

Video ini dari channel Ramiroy merangkum dan memberi ranking 23 buku psikologi dan self-help dalam format tierlist cepat. Buku-buku seperti "The 48 Laws of Power" oleh Robert Greene (tentang manipulasi dan kekuasaan dari sejarah), "The Laws of Human Nature" (psikologi untuk hindari iri hati), hingga stoikisme dari Epictetus dan Seneca (fokus pada kontrol diri dan visualisasi negatif). Ada juga buku seperti "The Subtle Art of Not Giving a..." untuk pilih prioritas, dan "The Rational Male" untuk dinamika dating. Deskripsi ini menyoroti pelajaran hidup, anekdot pribadi pembuat video (seperti pindah dari Suriah ke Dubai), dan campuran humor dengan saran untuk perbaikan diri.

The Web Application Hacker's Handbook

Rp 550.000

Video ini dari channel WhatIfTech merekomendasikan 5 buku esensial cybersecurity untuk pemula atau profesional IT. Termasuk "Hacking: The Art of Exploitation" yang ajarkan mindset hacker dengan contoh kode C dan buffer overflow, "The Web Application Hacker's Handbook" untuk serangan web seperti XSS dan SQL injection, serta "Metasploit: The Penetration Tester's Guide" untuk pengujian penetrasi. Ada juga "Applied Cryptography" untuk enkripsi dan "Blue Team Handbook" untuk respons insiden. Fokus pada keseimbangan ofensif-defensif, dengan saran praktik lab untuk bangun skill nyata di bidang keamanan digital.

pfSense Firewall Guide

Rp 300.000

Video ini dari channel The Social Dork menjelaskan 5 proyek cybersecurity gratis untuk home lab di 2026, menggunakan VirtualBox dan VM sederhana. Proyek seperti setup SIEM dengan WAZUH untuk analisis log, bangun portfolio dengan Sevalla (hosting gratis kredit $50), firewall dengan pfSense atau SafeLine WAF untuk blok serangan seperti SQL injection, CTF di Hack The Box/TryHackMe untuk latihan eksploitasi, analisis paket dengan Wireshark, dan proyek cloud security dari GitHub (misalnya IAM dan S3). Ideal untuk pemula yang ingin tambah resume dan skill praktis tanpa biaya besar, dengan link tutorial.

Linux Basics for Hackers

Rp 380.000

video dari channel David Bombal yang interview OccupyTheWeb tentang buku-buku hacking dan cybersecurity terbaik untuk 2023. Mereka bahas rekomendasi buku praktis untuk pemula sampai pro, fokus pada ethical hacking, pentesting, social engineering, dan topik spesifik seperti API atau connected cars. Buku-buku yang disebut termasuk "Linux Basics for Hackers" (untuk belajar Linux dari perspektif hacker), "Gray Hat Hacking" (referensi lengkap etika hacking), "Black Hat Python" (Python untuk pentesting), "Bug Bounty Bootcamp" (cari bug web), dan banyak lagi, dengan saran mulai dari dasar dan bangun karir. Cocok buat yang mau belajar cybersecurity hands-on, tapi OTW ingatkan jangan beli semua—mulai dari 1-2 buku aja. Ada juga diskusi soal buku outdated dan bahasa seperti Python vs Go.
